## ۱. ساخت ربات تلگرام

### مرحله ۱: گرفتن توکن

1. در تلگرام به [@BotFather](https://t.me/BotFather) پیام بدهید
2. دستور `/newbot` را بفرستید
3. یک **نام** برای ربات انتخاب کنید (مثلاً: `Iran News Bot`)
4. یک **یوزرنیم** انتخاب کنید که با `bot` تمام شود (مثلاً: `myirannews_bot`)
5. BotFather یک **توکن** به شما می‌دهد که شبیه این است:
   ```
   7812345678:AAH...long_string...
   ```
6. این توکن را ذخیره کنید — همیشه محرمانه نگه دارید!

### مرحله ۲: گرفتن شناسه کانال

دو حالت وجود دارد:

**حالت الف) کانال عمومی (با یوزرنیم):**
- مثلاً: `@my_iran_news`
- همین را در `.env` قرار دهید

**حالت ب) کانال خصوصی (بدون یوزرنیم):**
- ربات [@username_to_id_bot](https://t.me/username_to_id_bot) را استارت کنید
- هر پیام از کانال را فوروارد کنید تا شناسه عددی بدهد
- شناسه چیزی شبیه `-1001234567890` است

## ۲. ادمین کردن ربات در کانال

**این مرحله بسیار مهم است — در غیر این صورت ربات نمی‌تواند پیام بفرستد!**

1. وارد کانال خود شوید
2. روی نام کانال بالا کلیک کنید → **مدیریت کانال** (Manage Channel)
3. به **مدیران** (Administrators) بروید
4. **افزودن مدیر** (Add Administrator) را بزنید
5. یوزرنیم ربات خود را جستجو و انتخاب کنید
6. این دسترسی‌ها را **فعال** کنید:
   - ✅ ارسال پیام‌ها (Post Messages)
   - ❌ سایر دسترسی‌ها می‌توانند غیرفعال باشند

## ۳. گرفتن کلید رایگان Gemini

1. به [Google AI Studio](https://aistudio.google.com/app/apikey) بروید
2. با حساب گوگل وارد شوید
3. روی **Get API Key** یا **Create API Key** کلیک کنید
4. یک کلید شبیه این دریافت می‌کنید:
   ```
   AIzaSyXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX
   ```
5. کلید را ذخیره کنید

**محدودیت پلن رایگان:**
- ۱۵ درخواست در دقیقه
- ۱۵۰۰ درخواست در روز
- کاملاً کافی برای ربات ما

## ۴. راه‌اندازی روی کامپیوتر

### پیش‌نیاز
- پایتون 3.11 یا بالاتر ([دانلود از python.org](https://python.org))

### مراحل

```bash
# 1. وارد پوشه پروژه شوید
cd iran-news-bot

# 2. وابستگی‌ها را نصب کنید
pip install -r requirements.txt

# 3. فایل محیطی بسازید
copy .env.example .env
```

### فایل `.env` را ویرایش کنید:

```env
TELEGRAM_BOT_TOKEN=7812345678:AAH...your_token...
TELEGRAM_CHANNEL_ID=@your_channel_username
GEMINI_API_KEY=AIzaSy...your_key...
```

### تست کنید:

```bash
# بررسی صحت تنظیمات
python config.py

# تست اتصال تلگرام (یک پیام تست می‌فرستد)
python telegram_sender.py

# اجرای یک سیکل (دریافت + ارسال)
python bot.py --once
```

### اجرای دائم:

```bash
python bot.py
```

ربات حالا هر ۳۰ دقیقه اخبار عادی و هر ۵ دقیقه اخبار فوری را بررسی می‌کند.

## ۵. استقرار روی Render

برای اینکه ربات همیشه روشن بماند (حتی وقتی کامپیوتر خاموش است)، آن را روی Render مستقر کنید.

### مرحله ۱: کد را به GitHub ببرید

```bash
git init
git add .
git commit -m "Initial commit - Iran News Bot"
git branch -M main
git remote add origin https://github.com/USERNAME/iran-news-bot.git
git push -u origin main
```

⚠️ **مهم:** مطمئن شوید فایل `.gitignore` وجود دارد (وجود دارد) تا `.env` پوش نشود.

### مرحله ۲: ساخت سرویس در Render

1. به [render.com](https://render.com) بروید و ثبت‌نام کنید (با GitHub)
2. روی **New +** → **Background Worker** کلیک کنید
3. مخزن GitHub خود را انتخاب کنید
4. Render فایل `render.yaml` را به‌صورت خودکار می‌خواند
5. در بخش **Environment** این متغیرها را اضافه کنید:
   - `TELEGRAM_BOT_TOKEN` = توکن ربات شما
   - `TELEGRAM_CHANNEL_ID` = شناسه کانال شما
   - `GEMINI_API_KEY` = کلید Gemini شما

### مرحله ۳: استقرار

- **Create Background Worker** را بزنید
- چند دقیقه صبر کنید تا نصب انجام شود
- در بخش Logs می‌توانید فعالیت ربات را ببینید

✅ تمام! ربات حالا ۲۴ ساعته روشن است.

## ۶. عیب‌یابی

### مشکل: ربات پیام نمی‌فرستد

```
❌ ربات دسترسی به کانال ندارد. آیا ادمین است؟
```

**راه‌حل:**
- مطمئن شوید ربات **ادمین کانال** است (نه فقط عضو)
- `TELEGRAM_CHANNEL_ID` را با `@` شروع کنید برای کانال عمومی
- تست کنید: `python telegram_sender.py`

### مشکل: خطای Gemini

```
❌ خطای Gemini در ارزیابی خبر
```

**راه‌حل:**
- کلید API را دوباره چک کنید
- محدودیت رایگان را چک کنید (۱۵۰۰ در روز)
- ربات در صورت قطع Gemini به فیلتر کلمات کلیدی برمی‌گردد

### مشکل: اخبار دریافت نمی‌شوند

- اینترنت خود را بررسی کنید
- `python rss_parser.py` را اجرا کنید تا خطا را ببینید
- ممکن است یک منبع خاص موقتاً قطع باشد

### مشکل: خطای Markdown در تلگرام

ربات به‌صورت خودکار در صورت خطای Markdown، متن ساده می‌فرستد. اگر باز هم خطا داشت:
- قالب در `templates.py` را ساده‌تر کنید
- کاراکترهای `*` `_` `[` را کمتر کنید
